Simple toolkit to enable Block Editor for WooCommerce product edit page.

== Description ==

By default Block editor ( Gutenberg ) is not available in product page of backend for WooCommerce. It will helps you to enable gutenberg block editor for product.

== Frequently Asked Questions ==

= Are there any settings? =

No, there are no settings. Once activated, this plugin enabled Block editor ( Gutenberg ) for product.
